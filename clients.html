AMS

Association Maintenance Services provides professional maintenance staff all property maintenance needs from service repairs to light construction. We made an app that helps property managers submit service requests in a more efficient way. Now, when a property manager sees something that is broken at his property, he can immediately take a picture, record special instructions, and send it to AMS for prompt customer service.

C2D Technology

C2D Technology is developing leading edge deposition technologies for the automotive industry. Their proprietary deposition method allows for fuel efficiency that is far superior than the current leading method. We built the website to highlight their company and created animations to show how their process works. We also created marketing materials and edited demo videos for investors.

IMGW

IMGW is a GIS-based platform to manage risk mapping, security incident tracking, project management and basic service monitoring elements. This tool greatly enhances partners’ capacities for coordinated targeting, design and monitoring of interventions.
We developed this platform using different GIS providers such as Google Maps, Open Street Maps, Ushahidi, among others.

KalpTree Energy

KalpTree Energy is creating new battery architecture through electrochemical deposition. We redesigned their brand identity and their website. We’ve been working on creating marketing materials, investor presentations, videos, and print documents to keep up with the company’s growth.

Moneero

Moneero is a Startup dedicated to bring Bitcoins to the masses through social media, SMS and Bitcoin ATM’s (a.k.a. BTM). They gave us a BTM to use as a test machine and their backend API specifications. We took care of contacting the hardware manufacturers to ask them for API’s to talk to different components, being the most critical one the bill acceptor.

ROOTD

ROOTD is an athletics based networking platform where current student-athletes can search and apply for internships and jobs, connect with student-athlete alumni, and create profiles that showcase their athletic, academic and professional experiences.

We developed the Web App in Python + Django REST framework + MySQL + AngularJS and the system is currently publised on Amazon Elastic Beanstalk and S3.

US LADAR

US Ladar is developing a compact, low-power, and low-cost 3D vision sensor that will enable a new generation of autonomous vehicles and robotics. We built a software platform and configuration system that interfaces with their hardware sensors to allow customers to generate the most optimal configuration for their specific needs. We also created brochures, executive summaries, and product concept designs for acquiring customers and fundraising from investors.

VivaLnk

Vivalink is developing a low-cost digital tattoo that allows parents to measure their children’s temperatures. We built a mobile monitoring app that allows parents to record their child’s temperature by tapping their phones with the tattoo on their child. Parents can use this to track a child’s progress with an illness over time. The app used NFC to read and record temperature from the tattoo and graph it for the user and was featured at CES.

White Factor

White Factor is a 2-Factor authentication SaaS for companies that want to enable 2-Factor authentication to their users without having to implement a custom solution or use expensive ones from other providers. It uses Time Based One Time Passwords (TOTP) that change every 30 seconds. We developed the backend, frontend, password generation logic and the RESTful json API to enable integration with third party web apps.

zInspector

zInspector is a California real estate Startup that helps agents with property management. They had a mobile app that only sent photo’s through email so they asked us to develop a backend as well as a Web App. We created a JSON API, and developed the Web App in Python + Django + MySQL. The Web app uses lots of jQuery and Ajax and the system is currently published on Amazon Elastic Beanstalk. We also added a bunch of features to the iOS and Android mobile apps that were created using Corona SDK.
